Wikipedia Maze era uma versão do jogo que premiava pontos e medalhas tanto para a criação quanto para a solução de quebra-cabeças. Cada vez que um usuário resolvia um quebra-cabeça, ele ganhava pontos com base no número médio de cliques necessários para resolver o quebra-cabeça. Quanto mais difícil era o quebra-cabeça, maiores eram os pontos atribuídos. Cada quebra-cabeça poderia ser votado para cima ou para baixo por outros jogadores, com base no fato de eles gostarem ou não, o que atribuía pontos ao criador. Os jogadores também receberam distintivos por atingirem certos marcos. O site está extinto e seu URL de domínio foi adquirido e transformado em um redirecionamento para um site de malware.
